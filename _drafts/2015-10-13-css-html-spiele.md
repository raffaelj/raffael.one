Eine kleine Sammlung an schicken Browser-Spielen, bei denen man ganz nebenbei HTML und CSS lernen kann.

## online/am Computer

### Erase all kittens (en)
Bei [E.A.K.](https://eraseallkittens.com/) kannst du kleine, süße Kätzchen retten indem du Änderungen im Quellcode der Seiten vornimmst. Das Spiel und die Hilfe sind auf English.

### CSS Diner (en)
Beim [CSS Diner](http://flukeout.github.io/) kannst du hübsch animiert deine CSS-Selektoren- und vor allem deine CSS-Pseudo-Selektoren-Künste trainieren.
[Quellcode auf GitHub](https://github.com/flukeout/restaurant)
